Entitites:
i.Organizer-creates an event
ii.Attendee-enjoys the event.

Basic Structure:

An event will be organized. it will have a name , date and predtermined number of tickets .
Organizer will control these. he will also distribute the tickets.
Organizer can organize multiple events.

The Attendees will buy the ticket .

Functions:
1. Organize can create an event. Set the price , date and the number of tickets the event can have.

2. Attendees can buy tickets of multiple events. They can buy multiple tickets of same event. Also they can transfer ticket 
to another address.
